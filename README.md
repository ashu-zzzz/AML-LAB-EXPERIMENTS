## How to run

Pick an experiment folder and open the notebook or run the Python script.

### Notebooks

- Open the `.ipynb` file in Jupyter Notebook or VS Code.
- Run all cells.

### Python scripts

- Open a terminal in the experiment folder.
- Run: `python labX.py` (replace `X` with the lab number).

## Notes

- Some notebooks may require common ML libraries (numpy, pandas, matplotlib, scikit-learn).
- Data files are stored alongside the corresponding experiment where applicable.
